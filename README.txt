
# IT Help Desk Ticketing System (Google Forms + Sheets + Apps Script)

This mini-project simulates a simple IT help desk ticketing system using Google tools.

# Components:
- Google Form: for users to submit tickets
- Google Sheet: to store ticket data
- Google Apps Script: to automate ticket update notifications via email

# How It Works:
1. User submits issue using a Google Form.
2. Data is stored in the linked Google Sheet.
3. Admin updates status and message in Sheet.
4. Apps Script sends email to user automatically.

# Files Included:
- script.js — Apps Script code to automate email updates
- instructions.pdf — Full step-by-step guide
- sample_form_structure.txt — Example form questions
